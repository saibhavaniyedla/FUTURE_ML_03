# Customer Support Chatbot

## About the Project
This project is a **Customer Support Chatbot** built using **Dialogflow**. The chatbot acts as a virtual assistant that answers customer questions, provides support, and responds automatically. It’s similar to chatbots used by websites like Amazon, Flipkart, and Zomato.

## Features
- Greeting message: `"Hi! How can I help you?"`
- Answers for basic FAQs

## Skills Learned
- Understanding how chatbots work in websites and apps
- Designing conversational flows (greetings, help, fallback)
- Matching user questions to AI-generated answers

## Tools Used
- [Dialogflow](https://dialogflow.cloud.google.com/) – Free, no-code chatbot builder

## How to Use
1. Open the Dialogflow console.
2. Import your agent or create a new one.
3. Add intents for FAQs and greetings.
4. Test your chatbot using the Dialogflow interface.
5. (Optional) Connect to a website or Telegram for live deployment.
